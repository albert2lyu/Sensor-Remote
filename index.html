Sensor-Remote by misieq

Remote controll application using Gravity Sensor in HTC Diamond/Touch pro and Samsung Omnia.

Application simulates mouse moves and clicks (using virtual touchpad or GSensor), controlls Windows Media Player and keyboard via bluetooth serial or network. Project uses Windows Mobile Unified Sensor API from http://sensorapi.codeplex.com/ (included in repo).

Install

After client / server installation for serial connection: Desktop: - pair devices - in bluetooth settings add incloming COM port - in SensorServer settings choose serial connection - set the same port name as int bluetooth settings Mobile: - in bluetooth settings add outgoing COM port binded to your PC - in SensorRemote client choose serial connection - select the same port name as in bluetooth settings For network connections manually set ip address and port of server in both client and server app. Set the same connection password.
